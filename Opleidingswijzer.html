Opleidingswijzer

Opleidingen

Na deze opleidingen kan je alle kanten op gaan, zo kan je verder gaan studeren aan be HBO en als je dan nog verder wil gaan zelfs nog aan de universitaire opleiding. Je kan dan bij heel wat plekken terecht voor een volgende HBO opleiding hier een aantal voorbeelden:

  • - Hogeschool Rotterdam
  • - Haagse Hogeschool
  • - Hogeschool Leiden
  • - InHolland Haarlem

Beroepen

Het kan ook zijn dat je meteen wil beginnen met werken dit kan ook na de opleiding je hebt dan redelijk wat keus in banen, dit omdat er altijd veel vraag is vanuit het bedrijfsleven. Daarom hebben we hier een klein overzicht van welke banen je kan doen zodra je klaar bent met de opleiding.

  • Software developer
  • Backend developer
  • Frontend developer
  • Fullstack developer
  • Developer in een bepaalde software taal
  • Database specialist

Dit zijn dus een paar voorbeelden zodra je zou door studeren dan kan je meer gaan doen. Dit omdat er dan meer kennis gegeven wordt over de talen maar ook omdat er dan meer talen aangeboden worden, en je er dus meer voordeel uit kan halen met je baan.
